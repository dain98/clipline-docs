Clipline documentation lives in this repository and is published as a static Docusaurus site.

## What is here

- Markdown and MDX documentation under `docs/`
- Docusaurus site configuration in `docusaurus.config.ts`
- Sidebar generation in `sidebars.ts`
- GitHub Pages deployment through GitHub Actions

## Published site

The production site is configured for:

```text
https://dain98.github.io/clipline-docs/
```

After GitHub Pages is enabled for this repository, pushes to `main` build and publish the site automatically.
