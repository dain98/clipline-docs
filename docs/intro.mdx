Clipline is a lightweight, open-source game recorder for Windows. It records with a
no-injection capture path, keeps clips local by default, and adds useful timeline context such as
League of Legends match events and osu! set plays.

This documentation is split into three parts:

- **Desktop app**: install Clipline, record replays and sessions, review clips, trim, export,
  configure games, audio, storage, hotkeys, and optional Cloud uploads.
- **Reference**: compatibility, privacy and anti-cheat notes, troubleshooting, and keyboard
  shortcuts.
- **Clipline Cloud**: deployment guides for hosting the optional web library and sharing service.

## Current status

Clipline is nightly, testing-grade software. It is useful today as a tray recorder with replay
saves, full-session game recording, a local library, a first-party review player, lossless
keyframe-aligned trimming, multi-source audio, League match markers, osu! play blocks, and optional
Clipline Cloud upload support.

The Windows installer is not Authenticode-signed yet, so Windows SmartScreen can warn on first run.
The in-app updater is separately signed and checks update bundle signatures before installing.
