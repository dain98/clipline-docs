The global Save Replay hotkey is configurable in **Settings > Hotkeys**. Review-player shortcuts work
while a clip is open.

## Global

| Action | Default |
|---|---|
| Save Replay | `Alt+F10` |

The hotkey recorder accepts supported F-keys, middle mouse, Mouse4, Mouse5, and Ctrl/Alt/Shift
modifiers with supported keyboard keys.

## Playback

| Action | Shortcut |
|---|---|
| Play / pause | `Space` or `K` |
| Back / forward 5 seconds | `J` / `L` |
| Step 10 frames | Left / Right |
| Nudge 1 second | Shift + Left / Right |
| Nudge 0.1 second | `,` / `.` |
| Previous / next edit point | Up / Down |
| Jump to start / end | Home / End |

## Trim and markers

| Action | Shortcut |
|---|---|
| Set trim in / out | `I` / `O` |
| Next / previous marker | `M` / Shift + `M` |
| Slide the whole trim | Drag selection |
| Adjust trim edges | Drag edges |

## Zoom and view

| Action | Shortcut |
|---|---|
| Zoom in / out | `+` / `-` |
| Fit trim selection | `\` |
| Fit whole clip | Shift + `\` |
| Toggle snapping | `S` |
| Bypass snapping while dragging | Hold `Alt` |
| Zoom or pan timeline | Scroll; Shift + scroll pans |

## General

| Action | Shortcut |
|---|---|
| Close clip or leave the current view | `Esc` |
